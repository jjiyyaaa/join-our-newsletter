## Overview

![Design Preview](design\Desktop_1350px.jpg)

### Description
This is my solution for the **Join Our Newsletter** challenge from DevChallenges.io. I created a responsive subscription component that includes an email input field, a call-to-action button, and a custom logo.

### Built With
- Semantic HTML5 markup
- CSS3 (Custom Properties)
- Flexbox for layout
- Google Fonts (Poppins)
- Inline SVG for the logo

### What I Learned
In this project, I focused on:
- Using **Inline SVG** to render the logo directly without external images.
- Implementing **Hover & Focus effects** (Zoom in) using `transform: scale`.
- Styling form inputs to **hide the placeholder text** on click (`:focus`).
- Ensuring the layout looks good on Mobile, Tablet, and Desktop.
